# Chapin Bump In

Minimal MVP social game built with **Next.js (App Router)** + **TypeScript**, ready for local development and Vercel deployment.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build for production

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this repository to GitHub.
2. In Vercel, click **Add New Project** and import the repo.
3. Keep default settings (framework auto-detected as Next.js).
4. Click **Deploy**.

No extra configuration is required.
